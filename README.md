# Chess-GAME
Приложение на C++ (Qt Widgets), реализующее шахматную доску с расстановкой и перемещением фигур по правилам ООП.

## Реализовано
- Отображение доски 8×8 с координатной сеткой
- Классы фигур Pawn, Rook, Knight, Bishop, Queen, King, наследуемые от абстрактного класса Figure
- Вычисление и подсветка допустимых ходов каждой фигуры с учётом других фигур на доске
- Очерёдность хода между белыми и чёрными
- Проверка шаха и мата, завершение партии при мате
- Сохранение и загрузка расстановки фигур из текстового файла
- Автоматическое воспроизведение партии на основе файла расстановки и файла последовательности ходов

## Сборка
Открыть ChessBoard.pro в Qt Creator, собрать через qmake.
